fn read_packet(&mut self, buf: &mut [u8]) -> Option<usize>

Fill the buffer. Returns the number of bytes read. None or Some(0) indicates EOF.

Provided Methods

fn buffer_size() -> c_int

The buffer size is very important for performance. For protocols with fixed blocksize it should be set to this blocksize. For others a typical size is a cache page, e.g. 4kb.

Default: 4kb.
